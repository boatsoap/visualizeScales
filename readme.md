####Music Trainer v0.0.1

100% JavaScript fretboard showing notes in a particular scale.

##Current Instruments:
*Ukulele
*Guitar

##Current Scalse:
*Major
*Minor
*Dominant 7th
*Major 7th
*Minor 7th
*6th
*Augmented
*Diminished
*9th
*7b9

##To Use:
Clone or download the repo and drop `vizulizeScales.js` into any html page. Make sure there <div> with id="music-trainer"
